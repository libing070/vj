审计整改选择省份标准：

自2018年10月通报起，各公司如出现以下任意一种情形需重点进行整改：
1.通报当期疑似违规转售流量总值超过50TB且违规转售流量占本公司统付流量总量比例超过40%；
2.通报当期存在单一集团客户疑似违规转售流量超过10TB且占当月全集团违规转售流量总量比例超过5%；
3.存在同一集团客户有两期以上被判定为疑似违规转售流量。


审计问责选择省份标准：

1.自2018年10月通报起，对6个月中有3个月存在上述审计整改情形的公司予以审计问责；
2.对于持续审计结果核查不认真、不完整或反馈不真实的公司予以审计问责；
3.对于长期不传、漏传、错传流量相关基础数据从而影响审计结果的真实性、准确性及完整性的公司，将视情况进行审计问责。
